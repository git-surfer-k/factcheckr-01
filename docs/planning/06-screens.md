# Screens (화면 정의서)

> FactCheckr — 화면 목록 및 구성 요소
> 이 문서는 /screen-spec의 입력이 됩니다.

---

## B2C 앱 (일반 사용자)

### 하단 메뉴: 홈 | 랭킹 | 내기록 | 설정

---

### 1. 로그인/회원가입 (/auth)

- **진입**: 앱 최초 접속, 미인증 상태
- **요소**:
  - 로고 + 서비스 소개 문구
  - Email 입력 필드
  - OTP 인증 코드 입력 필드
  - Authentication Zero 연동
  - "OTP 발송" 버튼
  - "인증 완료" 버튼
- **이동**:
  - 인증 완료 → ② 홈
- **인증**: 불필요 (인증 화면 자체)

---

### 2. 홈 화면 (/)

- **진입**: 앱 시작, 하단 메뉴 "홈"
- **요소**:
  - [상단] 유튜브 URL 입력창 + "팩트체크" 버튼
  - [본문] 최근 팩트체크 결과 목록
    - 각 항목: 영상 썸네일, 제목, 채널명, 팩트 점수, 날짜
  - [하단] 네비게이션 바 (홈/랭킹/내기록/설정)
- **이동**:
  - URL 입력 후 버튼 → ③ 분석 로딩
  - 목록 항목 클릭 → ④ 리포트 상세
- **인증**: 필요

---

### 3. 분석 로딩 화면 (/analyze/:id)

- **진입**: 홈에서 URL 입력 후 팩트체크 버튼
- **요소**:
  - 영상 제목/썸네일 (가능한 경우)
  - 분석 진행 상태 표시 (단계별 체크리스트)
    - 영상 다운로드
    - 자막 변환
    - 주장 추출
    - 뉴스 대조
    - 리포트 생성
  - 예상 소요 시간
  - "취소" 버튼
- **이동**:
  - 분석 완료 → ④ 리포트 상세
  - 취소 → ② 홈
- **인증**: 필요

---

### 4. 리포트 상세 화면 (/reports/:id)

- **진입**: 분석 완료 후 자동 이동, 홈 목록 클릭, 내 기록 클릭
- **요소**:
  - 영상 제목 + 채널명 + 팩트 점수 (큰 숫자)
  - 5개 탭:
    - **탭1: 콘텐츠 요약** — 영상 핵심 내용 AI 요약
    - **탭2: 팩트 점수 + 상세분석** — 종합 점수, 점수 산출 근거, 분석 차트
    - **탭3: 주장별 검증 목록** — 각 주장 카드 (주장 원문, 판정, 확신도, 근거 뉴스 수)
    - **탭4: 과거 뉴스 링크** — 근거가 된 뉴스 기사 목록 (제목, 언론사, 날짜, URL)
    - **탭5: 채널 정보** — 채널 요약 (이름, 점수, 최근 팩트체크 수) + 채널 상세 이동 링크
  - "다운로드" 버튼 (유료/구독)
  - "AI 검증 한계" 안내 문구
- **이동**:
  - 다운로드 버튼 → ⑤ 다운로드 형식 선택
  - 채널 정보 탭 → 채널명 클릭 → ⑥ 채널 상세
  - 주장 카드 → 근거 뉴스 보기 → 외부 링크 (새 탭)
- **인증**: 필요

---

### 5. 다운로드 형식 선택 (모달 또는 /reports/:id/download)

- **진입**: 리포트 상세에서 다운로드 버튼
- **요소**:
  - 형식 선택: PDF / MD / DOCX / HWP
  - 구독 상태 확인
    - 구독 중 → 바로 다운로드
    - 미구독 → 구독 안내 + 결제 페이지 이동
  - "다운로드" 버튼
  - "취소" 버튼
- **이동**:
  - 다운로드 → 파일 저장
  - 미구독 시 → ⑨ 설정 (구독 관리)
  - 취소 → ④ 리포트 상세
- **인증**: 필요

---

### 6. 채널 상세 화면 (/channels/:id)

- **진입**: 리포트 상세 채널정보 탭, 채널 랭킹에서 클릭
- **요소**:
  - 채널 썸네일 + 이름 + 구독자 수
  - 신뢰도 점수 (큰 숫자) + 변화량 (▲▼)
  - 추이 그래프 (시계열 라인 차트)
  - 세부 지표: 정확도, 출처 인용률, 논조 일관성
  - 검증 이력 목록 (최근 팩트체크된 영상 목록)
    - 각 항목: 영상 제목, 점수, 날짜
  - 카테고리 태그
- **이동**:
  - 검증 이력 항목 클릭 → ④ 리포트 상세
- **인증**: 필요

---

### 7. 채널 랭킹 화면 (/ranking)

- **진입**: 하단 메뉴 "랭킹"
- **요소**:
  - 카테고리 탭: 정치 | 경제 | 사회 | 국제
  - 사용자 지정 태그 필터
  - 채널 랭킹 목록 (순위별)
    - 각 항목: 순위, 채널 썸네일, 채널명, 신뢰도 점수, 변화량
  - 정렬: 점수 높은순 / 점수 낮은순
- **이동**:
  - 채널 클릭 → ⑥ 채널 상세
- **인증**: 필요

---

### 8. 내 기록 화면 (/history)

- **진입**: 하단 메뉴 "내기록"
- **요소**:
  - 내가 검증한 영상 이력 목록 (최신순)
    - 각 항목: 영상 썸네일, 제목, 채널명, 팩트 점수, 검증 날짜
  - 검색/필터 (날짜, 점수 범위)
  - 비어있을 때: "아직 검증한 영상이 없어요" + 홈으로 이동 버튼
- **이동**:
  - 항목 클릭 → ④ 리포트 상세
- **인증**: 필요

---

### 9. 설정 화면 (/settings)

- **진입**: 하단 메뉴 "설정"
- **요소**:
  - 프로필 정보 (이메일, 이름)
  - 구독 관리 (현재 플랜, 만료일, 플랜 변경)
  - 알림 설정
  - 이용약관 / 개인정보처리방침
  - 로그아웃 버튼
  - 회원 탈퇴
- **이동**:
  - 구독 관리 → 구독 플랜 선택/변경 페이지
  - 로그아웃 → ① 로그인
- **인증**: 필요

## B2B 사이트 (기업용, 별도 URL)

---

### B-1. 로그인/회원가입 (/login)

- **진입**: B2B 사이트 접속
- **요소**:
  - 기업 로고 + B2B 서비스 소개
  - Email 입력 + OTP 인증
  - 기업 회원가입 폼 (기업명, 담당자, 이메일)
- **이동**:
  - 로그인 성공 → B-2 대시보드

---

### B-2. 대시보드 (/dashboard)

- **진입**: 로그인 후, 사이드 메뉴
- **요소**:
  - 구독 상태 카드
  - 최근 생성 리포트 목록
  - "새 리포트 요청" 버튼
  - 사용량 통계
- **이동**:
  - "새 리포트 요청" → B-3
  - 리포트 클릭 → B-4

---

### B-3. 기업정보 + 상품정보 입력 (/reports/new)

- **진입**: 대시보드 "새 리포트 요청"
- **요소**:
  - 기업명 (자동 입력)
  - 업종 선택
  - 광고 상품/서비스 정보 입력
  - 희망 타겟 카테고리 선택 (다중)
  - "AI 분석 시작" 버튼
- **이동**:
  - AI 분석 시작 → B-4 (분석 완료 후)

---

### B-4. AI 추천 채널 + 심층 리포트 (/reports/:id)

- **진입**: 기업정보 입력 후 분석 완료
- **요소**:
  - 추천 채널 리스트 (광고적합성 높은순)
    - 각 채널: 이름, 신뢰도 점수, 광고적합성 점수, 카테고리
  - 채널별 심층 분석
    - 팩트체크 이력 요약
    - 논조 분석
    - 브랜드 안전도 점수
  - 리포트 다운로드 (PDF)
- **이동**:
  - 채널 클릭 → 상세 분석 펼치기

---

### B-5. 결제/구독 관리 (/billing)

- **진입**: 사이드 메뉴
- **요소**:
  - 현재 구독 플랜
  - 결제 이력
  - 인보이스 다운로드
  - 플랜 변경/취소
  - 결제 수단 관리
- **이동**:
  - 플랜 변경 → 플랜 선택 페이지
