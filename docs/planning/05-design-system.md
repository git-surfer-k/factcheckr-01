# Design System

> Factis — 디자인 시스템

---

## 1. 브랜드 아이덴티티

### 1.1 브랜드 키워드

| 키워드 | 설명 |
|--------|------|
| **신뢰** | 검증된 정보, 투명한 근거 |
| **명확** | 복잡한 분석을 한눈에 |
| **객관** | 편향 없는 중립적 평가 |

### 1.2 브랜드 톤

- 차분하고 신뢰감 있는 톤
- 과장 없이 사실 중심
- 접근하기 쉬운 친근함 + 전문성

## 2. 색상 시스템

### 2.1 Primary Colors

| 판정 | 색상 | HEX |

| 대체로 거짓 | Orange | #EA580C |
| 거짓 (False) | Red | #DC2626 |
| 검증 불가 | Gray | #6B7280 |

## 4. 컴포넌트

### 4.1 점수 표시 (Score Badge)

```
┌──────────┐
│   72     │  ← 크기: Display, 색상: Verdict Color
│  /100    │  ← 크기: Small, 색상: Text Muted
│ 대체로   │
│ 사실     │  ← 크기: Body, 색상: Verdict Color
└──────────┘
```

### 4.2 주장 검증 카드 (Claim Card)

```
┌──────────────────────────────────────────┐
│ ● 사실  "실업률이 3.5%로 역대 최저다"     │
│                                          │
│ AI 확신도: 92%                            │
│ 근거 뉴스 3건                             │
│ [뉴스 보기]                               │
└──────────────────────────────────────────┘
```

### 4.3 채널 카드 (Channel Card)

```
┌──────────────────────────────────────────┐
│ [썸네일]  ○○뉴스                         │
│           구독자 120만                    │
│           신뢰도: 85점  ▲3               │
│           카테고리: 정치                   │
│           최근 팩트체크: 15건              │
└──────────────────────────────────────────┘
```

### 4.4 하단 네비게이션

```
┌──────┬──────┬──────┬──────┐
│  홈  │ 랭킹 │내기록│ 설정 │
│  🏠  │  📊  │  📋  │  ⚙️  │
└──────┴──────┴──────┴──────┘
```

### 4.5 분석 로딩 상태

```

│     ✅ 자막 변환 완료                     │
```

## 7. 감정/분위기

| 요소 | 방향 |
|------|------|
| 전체 분위기 | 뉴스룸 같은 신뢰감, 깔끔한 정보 전달 |
| 애니메이션 | 최소한 — 로딩 진행 상태 정도만 |
| 그림자 | 미세한 그림자 — 카드 부각용 |
| 모서리 | 8px 라운드 — 부드럽지만 단정한 느낌 |
