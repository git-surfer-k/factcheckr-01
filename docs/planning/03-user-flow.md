## B2C 앱 (일반 사용자)

### 메인 플로우: 팩트체크

```
[앱 진입]
    │
    ▼
┌──────────────┐
│  로그인 여부?  │

    │  │         │
    │  ▼         ▼

    │  │ 로그인/회원가입 │
    │  │ (AuthZ+OTP)   │
    │  └──────┬───────┘
    │         │
    ▼         ▼
┌──────────────────────────────────────┐
│  ② 홈 화면                            │
│  ┌─────────────────────────────────┐ │
│  │ [상단] URL 입력창                  │ │

               │
               ▼

               │
     ┌─────────┼──────────┐
     │         │          │
     ▼         ▼          ▼
[다운로드]  [채널 상세]  [홈으로]
     │         │
     ▼         ▼
┌──────────┐ ┌──────────────────────────┐
│ ⑤ 형식   │ │ ⑥ 채널 상세 화면          │
```

## B2B 사이트 (기업용, 별도 URL)

```
[B2B 사이트 접속]
    │
    ▼
┌──────────────┐
│ ① 로그인     │
│ (기업 계정)   │
└──────┬───────┘
       │
       ▼
┌──────────────────────────────────────┐
│  ② 대시보드                            │
│  - 최근 생성 리포트 목록                │
│  - 구독 상태                            │
│  - [새 리포트 요청] 버튼                │
└──────────────┬───────────────────────┘
               │
       "새 리포트 요청"
               │
               ▼
┌──────────────────────────────────────┐
│  ③ 기업정보 + 상품정보 입력             │
│  - 기업명, 업종, 브랜드                  │
│  - 광고 상품/서비스 정보                 │
│  - 희망 타겟 카테고리                    │
│  [AI 분석 시작] 버튼                    │
└──────────────┬───────────────────────┘
               │
               ▼
┌──────────────────────────────────────┐
│  ④ AI 추천 채널 + 심층 리포트           │
│  - 추천 채널 리스트 (적합도 순)          │
│  - 채널별 심층 분석                      │
│    · 팩트체크 이력                       │
│    · 논조 분석                           │
│    · 브랜드 안전도 점수                  │
│  - 리포트 다운로드                       │
└──────────────────────────────────────┘

[사이드메뉴]
    │
    ├── ② 대시보드
    ├── ③ 새 리포트 요청
    └── ⑤ 결제/구독 관리
            - 구독 플랜 선택
            - 결제 이력
            - 인보이스 다운로드
```

## 화면 간 이동 경로 요약

| 출발 | 도착 | 트리거 |
|------|------|--------|
| 홈 | 분석 로딩 | URL 입력 후 팩트체크 버튼 |
| 분석 로딩 | 리포트 상세 | 분석 완료 |
| 리포트 상세 | 다운로드 형식 선택 | 다운로드 버튼 (유료) |
| 리포트 상세 (채널정보 탭) | 채널 상세 | 채널명 클릭 |
| 홈 (최근 목록) | 리포트 상세 | 목록 항목 클릭 |
| 채널 랭킹 | 채널 상세 | 채널 클릭 |
| 채널 상세 (검증 이력) | 리포트 상세 | 이력 항목 클릭 |
| 내 기록 | 리포트 상세 | 항목 클릭 |
| 설정 | 구독 관리 | 구독 관리 메뉴 |
